### Installation

From source: 

    % cd xfwm4
    % ./autogen.sh
    % make
    % make install

From release tarball:

    % tar xf xfwm4-<version>.tar.bz2
    % cd xfwm4-<version>
    % ./configure
    % make
    % make install
